## Installation

```bash
npm i -g @nestjs/cli

nest new [projectName]]

select npm

npm i --save class-validator class-transformer  //DTO 유효성 검사기

npm i @nestjs/mapped-types // PartialType : 다른 class field 빌리기
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
